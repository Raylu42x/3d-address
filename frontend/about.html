Any point on Earth, in five memorable words

The 3D Word Address protocol turns a location into a short, human-readable address — and back — with a built-in typo check. It works the same for a front door, a drone waypoint, or a satellite.

installing-herd-chimera-washable-pendent-W  →  the Space Needle

How it works

Space around Earth is divided into a fixed grid of cells. Each word in an address selects one cell and then zooms into it — every added word narrows the location roughly 27,000×. The final letter is a checksum: it's computed from the words, so if you mistype one, the address is flagged and usually auto-corrected to the intended location.

There are two ways to write the same address. The human form is words with the checksum last (word-word-word-C). The machine form is the checksum first, followed by the raw cell numbers (C-4452-12644-6220-4824-5130) — handy for computers exchanging locations directly.

Precision is up to you

Use fewer words for an approximate area, more for pinpoint accuracy:

WordsCell sizeGood for
3~460 mNeighborhood / block
4~15 mA property or single home
5~51 cmA doorway (the everyday default)
6~1.7 cmA device or hand
7~0.6 mmRobotics / engineering

Fully 3DEvery address includes altitude, covering ~13 km underground to 2,000 km up — not just the surface.

Use it

In Python

pip install waddr
from waddr import encode_all, decode

info = encode_all(51.5074, -0.1278, alt_km=0.1, words=5)
print(info["words"])                 # dinosaur-epidural-usable-bingo-dusty-C
print(decode(info["words"])["lat"])  # 51.5074...

Over HTTP or self-hosted

Call the public API, or run the whole thing yourself with Docker. Full guides are in the GitHub repository.

Frequently asked questions

What is a 3D word address?

A short list of ordinary words — like installing-herd-chimera-washable-pendent-W — that names one small cell of space anywhere on or near Earth. Each word narrows the location about 27,000×, and the final letter is a checksum that catches typos. It's a human-friendly stand-in for GPS coordinates.

How is it different from latitude and longitude?

Coordinates are precise but hard to remember, read aloud, or type without errors. A word address carries the same precision in a few memorable words, and includes a built-in checksum so a single mistyped word is detected and usually auto-corrected.

What makes it 3D?

Every address encodes altitude as well as latitude and longitude, covering the whole near-Earth shell from about 13 km underground to 2,000 km up. The same system addresses a basement, a doorway, a drone waypoint, or a satellite.

Is it free?

Yes. The web app is free to use, the engine is open source (pip install waddr), and the code is MIT licensed. The official dictionary is shared under CC BY-ND so every implementation produces identical, interoperable addresses.

How accurate is a word address?

It depends on how many words you use: 3 words is about a city block (~460 m), 5 words is roughly a doorway (~51 cm), and 7 words is sub-millimeter. You choose the trade-off between length and precision.

Can I run my own copy or use it in my app?

Yes. Install the engine with pip install waddr, call the public HTTP API, or self-host the full web app with Docker. Because the dictionary and geometry are frozen, a self-hosted instance produces the exact same addresses as every other instance.
